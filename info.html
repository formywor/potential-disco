Overview

What’s new in V1?

ScriptNova Browser V1 is a Windows HTA-based launcher that wraps Google Chrome with a custom user agent, guarded access, and a clean status UI. This version focuses on compatibility with real websites while keeping your existing password system.

Improved website compatibility GeForce NOW works

Earlier versions used a Chromebook-style user agent that made some services think the browser was unsupported or “too automated.” V1 now uses a modern Windows Chrome user agent, which:

  • Works with NVIDIA GeForce NOW at nvidia.com.
  • Passes most “unsupported browser” checks that blocked older versions.
  • Looks like a normal, up-to-date Chrome install on Windows.

The launcher still calls the real Chrome.exe on your machine, it just tells sites to treat it as a supported configuration.

Password-protected launcher

Access to the launcher is still locked behind a remote password list:

  • Valid passwords are stored remotely in ERROR ADMIN ONLY TEXT.
  • On every run, V1 downloads that list and checks the code you enter.
  • If the password isn’t on the list, the launcher closes and Chrome is not opened.
Note: Info shown here is high level only. Actual passwords are never shown on this page and are still managed remotely.

Version-locked for safety

V1 includes a version check against your site so you can force updates:

  • On start, the launcher reads browser/version.txt from your server.
  • If that file doesn’t say 17, the launcher stops and tells the user to download the new build.
  • This prevents old, out-of-date launchers from being used once you ship a new version.

Launch behavior

After a correct password and version check:

  • Any existing chrome.exe process is closed to avoid conflicts.
  • Chrome is launched with a custom user agent so sites see it as a supported browser.
  • Multiple tabs can be opened on launch, including:
    • Your Instagram profile(s).
    • The Gomega AI page at https://gomega.watch/ai.

Visual + UX updates

  • Glassmorphism-style launcher UI with live status text.
  • Spinner and status line so users know what’s happening:
    • Integrity / version checks.
    • Password loading.
    • Browser launch.
  • Soft integrity check: if the launcher can’t read its own file, it warns but keeps going instead of crashing.

What V1 does not do

  • It does not bypass GeForce NOW requirements or any other service’s rules.
  • It does not change how Chrome itself executes code or installs extensions.
  • It does not manage your GeForce NOW account or login in any way.

V1 simply launches the browser with a user agent that many services expect and trust.

For users

How to use ScriptNova Browser V1

  • Run the ScriptNova Browser V1 launcher file.
  • Wait for the status text to say it’s checking version and loading passwords.
  • Enter your launcher password when prompted.
  • If the password is valid, Chrome will open with the ScriptNova configuration.
  • From there, you can visit sites like GeForce NOW and others as normal.
